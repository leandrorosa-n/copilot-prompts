## 💻 Sobre o Projeto

Construção de uma estrutura eficiente de agentes generativos para identificar o seu perfil em tecnologia, e aprimoramento na evolução com estratégias por um Roadmap.

## 📚 Pré-requisitos de Habilidades e Níveis de Conhecimento

Antes de ingressar neste conteúdo, é necessário possuir conhecimento prévio nas seguintes áreas:

  - IA Generativa | Básico
  - Mapeamento de carreira | Básico
  - Lógica de programação | Básico


## 🛠️ Habilidades e Sub-habilidades que vamos aprender neste conteúdo

- Motor de perfil estruturado
- Análise de oportunidades


## 🎯 Objetivos e Resultados Esperados

Após a conclusão do projeto, os estudantes estarão aptos a:

- Entender com precisão o perfil de tecnologia para ser inserido no mercado de trabalho
- Construir um Agente de IA por prompt para ajudar nas decisões
